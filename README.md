# The Frontend of KING Pacman site

## Release date
May 26, 2022

## Environment
- `Node.js v14.17.0`

## Stack
- **Framework:** `React.js v18.1.0`
- **Theme:** `MUI v5.8.0`

## How to run the game.
1. Please open terminal window in the root directory.
2. Please run command `npm run start` in it.
